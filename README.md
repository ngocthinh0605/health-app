# Health App

A web application for tracking your health.

## Features

- pages: Home, YourRecord, Column, NotFound
- components: Header, Footer, Logo, Navigation, Menu, GoToTopButton, Button, Typography

## Tech Stack

- React
- Vite
- TypeScript
- Tailwind CSS

## Prerequisites

- Node.js v18.17.1
- npm v10.1.0
